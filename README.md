# assignment1_rt

This package is part of a ROS workspace.

## Overview

`assignment1_rt` contains code and resources for Assignment 1 in Real-Time Robotics. It is designed to be used within a ROS (Robot Operating System) environment.

## Installation

Clone this repository into your ROS workspace `src` directory:

```bash
cd ~/ros_ws2/src
git clone <repository-url> assignment1_rt
cd ..
catkin_make
```

## Usage

Source your workspace and launch the package nodes as required:

```bash
source devel/setup.bash
roslaunch assignment1_rt <launch_file>.launch
```

## File Structure

- `src/` - Source code for nodes and scripts
- `launch/` - ROS launch files
- `README.md` - This documentation
